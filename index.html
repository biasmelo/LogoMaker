LogoMaker

O software LogoMaker permite que usuários comuns gerem logotipos. É um software particularmente útil para empreeendedores porque de repente a pessoa tem uma ideia de um negócio mas às vezes a pessoa não tem habilidade com edição de imagem.
Nesse caso a pessoa usa nosso software para gerar um logotipo e isso ajuda a dar uma cara para o negócio porque o logomaker sugere cores e com isso novas ideias de layout aparecem.

O logomaker é pensado especificamente para programadores por que muitas vezes o cliente surge sem saber exatamente como ele quer o site ou ou app.
Quais as cores apropriadas... qual o tom...

Se o produto deve apresenar uma imagem séria ou corriqueira. Nesse caso o programador gera alguns logotipos provisórios para o site do cliente. Um placeholder. Gerar um logotipo para o site do cliente dá ao programador idéias de como o site deve ficar. É meio que automático.
O LogoMaker é gratuíto e open-source. Ele foi programado com a linguagem javascript e está disponível para programadores (e outros tipos de usuário) no site oficial e no repositório oficial no github.
